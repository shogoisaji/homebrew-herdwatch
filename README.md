# homebrew-herdwatch

Homebrew tap for [HerdWatch](https://github.com/shogoisaji/HerdWatch).

## Install

```bash
brew tap shogoisaji/herdwatch
brew install --cask herdwatch
```

or directly:

```bash
brew install --cask shogoisaji/herdwatch/herdwatch
```

## Update

The Cask is updated automatically when a new HerdWatch release is published
(via `workflow_dispatch` from the main repository's Release workflow).
